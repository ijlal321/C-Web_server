CivetWeb Common HTTP API Summary
===============================

1. mg_printf
   - Print formatted (text or HTML) data to the HTTP response, like printf for files/console.
   - Usage: mg_printf(conn, "Hello %s!", name);
   - Example: mg_printf(conn, "<h1>Welcome, %s!</h1>", username);
   - Think: printf, but output goes to the HTTP client/browser.

2. mg_write
   - Write raw data (buffer, string, etc.) to the HTTP response.
   - Usage: mg_write(conn, data, data_len);
   - Example: mg_write(conn, "OK", 2);
   - Use for binary or already-formatted data.

3. mg_send_chunk
   - Send a chunk of data (for chunked transfer encoding).
   - Usage: mg_send_chunk(conn, data, data_len);
   - Only use after sending headers with "Transfer-Encoding: chunked".

4. mg_send_file
   - Send a file with HTTP headers (Content-Type auto-detected).
   - Usage: mg_send_file(conn, "file.txt");
   - (Superseded by mg_send_mime_file, but still useful for quick file serving.)

5. mg_send_file_body
   - Send a file body only (no headers!).
   - Usage: mg_send_file_body(conn, "file.txt");
   - Must send your own headers first (e.g., mg_send_http_ok).
   - Returns <0 on error.

6. mg_send_http_error
   - Send an HTTP error response (with status code and formatted message).
   - Usage: mg_send_http_error(conn, 404, "Not found: %s", path);

7. mg_send_http_ok
   - Send HTTP 200 OK header (set Content-Type and content length).
   - Usage: mg_send_http_ok(conn, "text/html", file_size);
   - If content_length == -1, uses chunked transfer encoding.
   - After this, use mg_write, mg_printf, or mg_send_chunk for body.

Quick Reference:
----------------
mg_printf(conn, "<html>...</html>");         // Print formatted response (like printf)
mg_write(conn, data, len);                   // Write raw data
mg_send_http_ok(conn, "text/html", -1);     // Start chunked response
mg_send_chunk(conn, data, len);              // Send chunk
mg_send_file(conn, path);                    // Send file with headers
mg_send_file_body(conn, path);               // Send file body only
mg_send_http_error(conn, 500, "Oops");      // Send error
